# Spring Project
学校综设C课程，借以学习Spring与Java。

## 运行
**`cd .\digitalworld\avc-digitalworld-main\`后才可执行以下命令**
启动：`mvn spring-boot:run`
编译：`mvn compile`
clean：`mvn clean`

## environment(recommend)
```
Apache Maven 3.6.3
Java version: 11.0.6
```
