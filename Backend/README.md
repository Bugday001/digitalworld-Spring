# Spring Project
*后端部分，采用Spring实现。*

## 运行
**`cd .\digitalworld\avc-digitalworld-main\`后才可执行以下命令**
启动：`mvn spring-boot:run`
编译：`mvn compile`
clean：`mvn clean`

## Communication
- [X] Http
- [X] WebSocket
- [X] UDP

## environment(recommend)
```
Apache Maven 3.6.3
Java version: 11.0.6
```
